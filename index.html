祝贺你

你已经成功在服务器上安装了串流服务。当你看到该页面的时候，你可以看到JPG格式的图片串流，该图片来自UVC摄像头的一个捕捉示例。安装的串流服务包括这些示例页面而且你可以自定义页面的外观和内容。

开发MJPEG-Streamer的初衷是为了使用基于Linux-UVC标准的网络摄像头搭建一个易于使用和资源友好型的在线串流网站。虽然该项目的前任 uvc-streamer 能够很好地工作，但是接下来我想加入一些新鲜的想法。例如，插件可被用来处理捕捉的图像。其中一个输入插件将图像复制成为一个全局变量，接下来多个输出插件都有该变量的访问权限。比如现在你看到的这个网页就是由 output_http.so 插件来驱动。

显示在这里的图片来自内置输入插件的抓取结果。该HTTP请求包括GET参数 action=snapshot。这个图片的显示要求图像输入的一个画面。为了显示另外一个图片示例，请点击左侧图片。

关于该示例

为了使用任意浏览器都可以看到该串流，你可能需要尝试 javascript 或者 java 子页面。火狐浏览器能够直接显示MJPG-Streamer的串流。
